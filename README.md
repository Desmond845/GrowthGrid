# GrowthGrid 🌱

A beautiful, free personal growth tracking app that helps you build better habits, reflect on growth and track your progress daily.

## Features
- 📊 Track multiple growth aspects (health, coding, learning, etc.)
- 📝 Daily journal entries with smart organization  
- 🎯 Goal setting and progress monitoring
- 💾 Local storage - your data stays private
- 📱 Responsive design - works on all devices
- 🆓 100% Free and open source

## Live Demo
Visit: https://desmond845.github.io/GrowthGrid/

## Technology
Built with vanilla JavaScript, HTML5, CSS3 with modern features like:
- Local Storage API
- CSS Grid & Flexbox
- Responsive Design
- Progressive Web App features
- Data Export
